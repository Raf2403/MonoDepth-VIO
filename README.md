# MonoDepth-VIO (IMU Publisher Branch)

Ветка imu-publisher добавляет в проект MonoDepth-VIO готовый программный модуль для работы с данными инерциальных датчиков (IMU). Он содержит скрипты для сбора и парсинга сырых данных с датчиков смартфона (акселерометра, гироскопа, магнитометра), их визуализации в виде графиков для первичного анализа, а также конвертации в формат ROS2 для публикации в стандартные топики. Этот пакет создаёт реалистичный и готовый к использованию источник сенсорной информации, который важен для тестирования и отладки алгоритмов визуально-инерциальной одометрии (VIO), разрабатываемых в основном проекте. Таким образом, ветка предоставляет полный инструментарий — от сбора и проверки данных на устройстве до их интеграции в ROS2-экосистему, что значительно ускоряет цикл разработки и валидации навигационных систем.

## Особенности
- Монокулярная визуальная одометрия с оценкой глубины.
- Интеграция IMU для улучшения точности и масштаба.
- Публикация IMU-топика в ROS (предположительно ROS2).
- Поддержка последовательностей изображений и синхронизированных IMU-измерений.

## Пояснения
- imu_csv_publisher/csv_imu_publisher.py — основная ROS2-нода, публикующая данные из CSV-файлов в топики /imu/data и /imu/mag.
- scripts/ — вспомогательные скрипты: сбор и парсинг данных со смартфона, визуализация графиков IMU, конвертация в ROS2 Bag.
- launch/ — launch-файлы для удобного запуска ноды с параметрами.
- package.xml, setup.py — конфигурационные файлы ROS2/Python пакета.
- ros_imu_data.csv — пример данных для тестирования.

## Установка
1. Клонируйте ветку:
git clone -b imu-publisher https://github.com/Raf2403/MonoDepth-VIO.git
cd MonoDepth-VIO

2. Установите зависимости:
sudo apt install ros-noetic-cv-bridge ros-noetic-image-transport
pip install tensorflow opencv-python

3. Соберите:
catkin_make
source devel/setup.bash
